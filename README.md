## Mô tả
Repository này chứa các bài tập thực hành môn **Kiểm thử phần mềm**,
bao gồm các bài tập về **JUnit**, **viết test case** và **kiểm thử đơn vị**.

## Công nghệ sử dụng
- Java
- Maven
- JUnit 5
- IntelliJ IDEA

## Cách chạy project
1. Mở project bằng IntelliJ IDEA
2. Chạy file test trong thư mục `src/test/java`
3. Xem kết quả kiểm thử
